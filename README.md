# Movie Finder Application

Movie Finder is a React-based web application that allows users to search and discover movies using external movie APIs.

- View Movie Ratings
- View Release Dates
- View Genres
- View Cast Details
- Movie Posters
- Responsive Design
- Real-Time API Integration

- JavaScript
- HTML5
- CSS3
- REST APIs
